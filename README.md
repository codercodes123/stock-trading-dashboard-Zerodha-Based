# 📈 Stock Trading Dashboard (Zerodha Clone)

A full-stack stock trading dashboard inspired by Zerodha.  
It allows users to **sign up, log in, manage portfolios, track watchlists, view holdings, and analyze stock data** with charts.  

---

## 🚀 Features
- 🔑 **Authentication System** – User signup & login with unique ID and email.  
- 📊 **Watchlist** – Add and track stocks with real-time updates.  
- 💼 **Holdings & Positions** – Simulated portfolio management.  
- 💸 **Buy & Sell Actions** – Interactive buy/sell window for stocks.  
- 📉 **Charts & Analytics** – Doughnut chart and visual portfolio analysis.  
- 📂 **Modular Codebase** – Separated into frontend, backend, and dashboard.  

---

## 🛠️ Tech Stack
- **Frontend**: React.js, Material-UI, Axios  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB (or any DB you configure)  
- **Charts**: Chart.js / Recharts  
- **Authentication**: JWT-based auth
